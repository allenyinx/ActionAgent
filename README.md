# airgent
webdriver agent


## Features
* Crawler for deep links
* Headless Chromium
* WebDriver
* Report
* Kafka message consumer
* Screenshots
* Log, reports, console info
* Produce kafka message to sitemap
* Parse and exec kinds of actions

Install
=======

The quick way::

    docker run -p 5900:5900 airta/airgent:latest
